# 랜딩페이지 이관 가이드

## 개요
현재 테스트용 구글시트/Apps Script가 개발자 계정에 연결되어 있습니다.
실제 운영을 위해 클라이언트 계정으로 이관이 필요합니다.

---

## 이관 절차

### 1단계: 구글시트 생성
1. 클라이언트 구글 계정으로 로그인
2. [Google Sheets](https://sheets.google.com) 접속
3. 새 스프레드시트 생성
4. 시트 탭 이름을 `응답`으로 변경 (하단 탭 우클릭 > 이름 바꾸기)

### 2단계: Apps Script 설정
1. 스프레드시트에서 **확장 프로그램 > Apps Script** 클릭
2. 기본 코드 삭제
3. 아래 코드 전체를 복사하여 붙여넣기
4. **SHEET_ID 수정**: 스프레드시트 URL에서 ID 복사하여 교체
   - URL 예시: `https://docs.google.com/spreadsheets/d/{이 부분이 SHEET_ID}/edit`
5. 저장 (Ctrl+S)

```javascript
const SHEET_ID = '여기에_시트ID_입력'; // 반드시 수정!
const SHEET_NAME = '응답';
const BUZZVIL_API_URL = 'YOUR_BUZZVIL_API_URL';
const BUZZVIL_API_KEY = 'YOUR_BUZZVIL_API_KEY';

function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    saveToSheet(data);
    // callBuzzvilApi(data); // 버즈빌 연동 시 주석 해제
    return createResponse({ success: true, message: '신청이 완료되었습니다.' });
  } catch (error) {
    Logger.log('오류 발생: ' + error.message);
    return createResponse({ success: false, error: error.message });
  }
}

function doGet(e) {
  return createResponse({ status: 'ok', message: '정상 작동 중' });
}

function saveToSheet(data) {
  const sheet = SpreadsheetApp.openById(SHEET_ID).getSheetByName(SHEET_NAME);
  if (!sheet) throw new Error('시트를 찾을 수 없습니다: ' + SHEET_NAME);
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(['제출일시', '이름', '연락처', '이메일', '개인정보동의', '버즈빌응답']);
  }
  sheet.appendRow([new Date(), data.name || '', data.phone || '', data.email || '', data.privacy ? 'Y' : 'N', '']);
  Logger.log('저장 완료: ' + data.name);
  return sheet.getLastRow();
}

function callBuzzvilApi(data) {
  // 버즈빌 API 명세에 맞게 수정 필요
  const payload = { user_id: data.phone, mission_type: 'form_submit' };
  const options = {
    method: 'POST',
    contentType: 'application/json',
    headers: { 'Authorization': 'Bearer ' + BUZZVIL_API_KEY },
    payload: JSON.stringify(payload),
    muteHttpExceptions: true
  };
  const response = UrlFetchApp.fetch(BUZZVIL_API_URL, options);
  return JSON.parse(response.getContentText());
}

function createResponse(data) {
  return ContentService.createTextOutput(JSON.stringify(data)).setMimeType(ContentService.MimeType.JSON);
}

function testSaveToSheet() {
  saveToSheet({ name: '테스트', phone: '010-1234-5678', email: 'test@test.com', privacy: true });
}
```

### 3단계: 테스트 실행
1. 함수 선택 드롭다운에서 `testSaveToSheet` 선택
2. 실행 버튼(▶) 클릭
3. 권한 승인 (구글 계정 선택 > 고급 > 안전하지 않은 앱으로 이동 > 허용)
4. 구글시트에 테스트 데이터 저장 확인

### 4단계: 웹앱 배포
1. **배포 > 새 배포** 클릭
2. 유형: **웹 앱** 선택
3. 설정:
   - 설명: `랜딩폼 백엔드 v1`
   - 실행할 사용자: **나**
   - 액세스 권한: **모든 사용자**
4. **배포** 클릭
5. **웹앱 URL 복사** (개발자에게 전달)

---

## 개발자 작업 (URL 수신 후)

클라이언트로부터 웹앱 URL을 받으면:

1. `script.js` 파일 열기
2. `APPS_SCRIPT_URL` 값을 새 URL로 교체:
```javascript
const APPS_SCRIPT_URL = '클라이언트가_전달한_URL';
```
3. Cloudflare Pages 재배포

## 주의사항

1. **SHEET_ID를 반드시 수정**해야 합니다. 안 하면 권한 오류 발생
2. 권한 승인 시 "안전하지 않은 앱" 경고가 나오는 건 정상입니다 (구글 검증 전)
3. 코드 수정 후에는 **새 배포**를 해야 반영됩니다 (배포 관리 > 새 배포)
4. 버즈빌 API 연동은 명세서 수령 후 별도 안내 예정
